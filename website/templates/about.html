About


CarZam is an application developed as part of a Capstone project in the MIDS program at the I-School at UC Berkeley. The application leverages deep learning neural network models to detect and predict the make and model of various cars sold in the US. The model was trained on an extensive dataset that combined public datasets as well as images sourced from webscraping automobile sales sites.

Problem Statement


There are existing tools to detect cars in images and videos but no comprehensive tool for also detecting make and model of the cars. For security and surveillance applications, the additional value of detecting make and model can help quickly narrow down and focus analysis. For traffic and commercial applications this tool can be extended to count makes and models of cars passing through intersections, tool gates, or on highways.

Datasets

  • Webscraped Data
  • Stanford Data
  • VMMR DB

Model Training


CarZam is based on the YOLOv5 Object Detection Model. We custom trained X-large YoloV5 model with our own custom car make dataset which included 58357 images classified into 81 unique car models.We used k-folds cross validation with 5 folds. Each fold trained for 70 Epochs (about 17 hours per fold on a dual V100 GPU system)

Results


Our model has shown to deliver about 95% precision and about 99% recall across folds. To learn more about the results of our training of the Carzam model please refer to our Results page.

About


CarZam is an application developed as part of a Capstone project in the MIDS program at the I-School at UC Berkeley. The application leverages deep learning neural network models to detect and predict the make and model of various cars sold in the US. The model was trained on an extensive dataset that combined public datasets as well as images sourced from webscraping automobile sales sites.

Problem Statement


There are existing tools to detect cars in inages and videos but no comprehensive tool for also detecting make and model of the cars. For security and surveillance applications, the additional value of detecting make and model can help quickly narrow down and focus analysis. For traffic and commercial applications this tool can be extended to count makes and models of cars passing through intersections, tool gates, or on highways.
